What makes it different

Built around the way you actually read.

Zero friction

No hotkey, no right-click, no copy-paste. The translation appears the moment your cursor touches a word. Move on — it disappears.

On-device AI

Uses Chrome's built-in Gemini Nano for private, offline-capable translation with no API keys needed. Falls back to cloud engines when needed.

Grammar overlay

Sentence mode color-codes each word by grammatical role — nouns, verbs, adjectives. Helps you absorb the grammar, not just the words.

Smart detection

Skips buttons, icons, menus, and UI elements — only triggers on actual readable text. You get translations where they make sense.

100+ languages

Automatic source detection or manually fix a language pair. Works in both directions. Supports everything from Spanish to Japanese to Arabic.

Your call, every site

Toggle translation per domain with one click. Customize the tooltip: font, size, blur, border radius. Enable only when you want it.

How it works

Up and running in two minutes.

1
Install

Add the extension to Chrome from the Web Store. No account, no signup.

2
Set your language

Pick your target language once. Auto-detection handles the rest.

3
Browse normally

Visit any page with foreign text. Transover sits quietly in the background.

4
Hover to read

Point at any word or sentence. The translation appears instantly, then fades when you move on.
